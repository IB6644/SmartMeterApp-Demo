# CEM6 誤結線検出アプリ - インタラクティブデモサイト

React Native製モバイルアプリのUIを完全再現したブラウザベースのインタラクティブデモサイトです。

## 🎯 プロジェクト概要

このデモサイトは、大崎電気工業株式会社のCEM6シリーズ電力量計向け誤結線検出モバイルアプリのUIを、Webブラウザ上で完全に体験できるインタラクティブなデモンストレーションです。

実際のBluetooth通信は行いませんが、アプリの全機能とユーザーインターフェースを**スマートフォンとタブレットの両方**で確認できます。

## 🌐 デモサイトURL

**👉 https://ib6644.github.io/SmartMeterApp-Demo/**

## ✨ 主要機能

### 🎨 完全なUI再現

元のReact Nativeアプリ（SmartMeterMobileApp）の以下の機能を完全再現：

#### 1️⃣ **ホーム画面**
- CEM6誤結線検出アプリのメインメニュー
- 3つの主要機能へのナビゲーション
- 使用前チェックリスト表示
- 企業カラー（#aa0000）を使用したブランディング

#### 2️⃣ **計器情報取得画面**
- Bluetooth接続シミュレーション（TP-BTデバイス）
- 計器基本情報の表示
  - 計器ID、製造No、形名
  - 相線式、定格電圧、定格電流
- 計量値情報の表示
  - 有効電力量、無効電力量、現在電力
- Excel出力への遷移

#### 3️⃣ **誤結線確認画面**
- Bluetooth接続シミュレーション
- 配線チェック設定
  - 相線式選択（三相3線式/三相4線式/単相2線式/単相3線式）
  - 定格電圧選択（200V/100V/400V）
  - 接続機器選択（模擬負荷/配線チェック機）
- 判定結果表示
  - 総合判定（正常/異常）
  - 検出パターン番号
  - 測定値（P1/P3 有効・無効電力）
- Excel出力への遷移

#### 4️⃣ **計量値確認画面**
- Bluetooth接続シミュレーション
- リアルタイム計量タイマー
- 計量開始/停止機能
- 計量記録表示
  - 開始値、終了値、差分
  - 計量時間
- Excel出力への遷移

#### 5️⃣ **Excel出力画面**
- 3つの出力フォーマット選択
  - **フォーマットA** - きんでん社用（詳細情報含む）
  - **フォーマットB** - 大崎標準（標準的な情報）
  - **フォーマットC** - スパイダープラス連携用
- 追加フィールド入力
  - 計量対象名、設置場所、備考
- 出力ファイル名生成（タイムスタンプ付き）

### 📱 デバイスエミュレーター機能

#### スマートフォンモード
- **画面サイズ**: 375 x 667px（iPhone 8相当）
- **タッチ最適化UI**: 大きなボタン、明確なフィードバック
- **縦向き表示**: モバイルファーストデザイン

#### タブレットモード
- **画面サイズ**: 768 x 1024px（iPad相当）
- **広々としたレイアウト**: より多くの情報を同時表示
- **横向き対応**: 大画面での使用感確認

#### デバイスフレーム
- リアルなAndroidデバイス風フレーム
- ステータスバー（時計、信号、バッテリー表示）
- ダークモードのデバイス筐体

### 🎭 インタラクティブ機能

#### リアルな動作シミュレーション
- **Bluetooth接続**: 接続中アニメーション → 2秒後に接続完了
- **データ取得**: ローディング表示 → 1.5-2.5秒後に完了
- **計量タイマー**: リアルタイムカウントアップ（分:秒表示）
- **画面遷移**: スムーズなアニメーション効果

#### 状態管理
- 接続状態の視覚的表示
  - 🔴 未接続（グレー）
  - 🟠 接続中（オレンジ、パルスアニメーション）
  - 🟢 接続済み（グリーン、光るエフェクト）
  - ❌ エラー（レッド）
- ボタンの有効/無効切り替え
- データ表示の動的制御
